รายละเอียดคลาสเรียน

เรียนรู้การสร้าง Web Application ตั้งแต่เริ่มต้นจนสามารถ Deploy ขึ้น Server ได้จริง โดยใช้เทคโนโลยียอดนิยมอย่าง React สำหรับ Frontend และ Node.js (Express) สำหรับ Backend คอร์สนี้เน้นลงมือปฏิบัติจริง (Workshop-based) เหมาะสำหรับผู้เริ่มต้นที่มีพื้นฐาน HTML/JS มาบ้าง

สิ่งที่คุณจะได้เรียนรู้:
  • พื้นฐาน React Components, Hooks และ State Management
  • การสร้าง RESTful API ด้วย Node.js และ Express
  • การเชื่อมต่อ Database (MongoDB)
  • การทำ Authentication (Login/Register)
